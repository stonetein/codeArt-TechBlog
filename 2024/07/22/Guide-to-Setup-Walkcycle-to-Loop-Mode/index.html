Guide to Setup Walkcycle to Loop Mode

Step-by-Step Guide to Setup Walkcycle to Loop Mode and Create Plane with Subdivisions

Step 1.Import Walkcycle Animation

首先，打開Maya並導入你的Walkcycle動畫。

Step 2.Setup Walkcycle to Loop Mode

設置Alembic動畫循環模式

1. 選擇Alembic節點

  • 在Outliner面板中，選擇包含Alembic動畫的對象。在這個例子中，選擇的“walkCycle_p01_AlembicNode”。

2. 打開Alembic屬性

  • 在右側屬性面板中，找到「Alembic Attributes」部分。
  • 確保「AlembicNode」已被選中，並展開「Alembic Attributes」部分。

3. 設置循環類型

  • 在「Cycle Type」選項中，選擇合適的循環模式。
  • 在這個例子中，選擇「Loop」模式，使動畫無縫循環播放。

4. 設置動畫參數

  • 設置動畫的開始幀(Start Frame)和結束幀(End Frame)，在這個例子中分別設置和35。
  • 確保其他參數如速度(Speed)和偏移量(Offset)設置正確，以實現所需的動畫效果。設置Alembic動畫循環模式

5. 選擇Alembic節點

  • 在Outliner面板中，選擇包含Alembic動畫的對象。在這個例子中，選擇的是 “walkCycle_p01_AlembicNode”。

6. 打開Alembic屬性

  • 在右側屬性面板中，找到「Alembic Attributes」部分。
  • 確保「AlembicNode」已被選中，並展開「Alembic Attributes」部分。

7. 設置循環類型

  • 在「Cycle Type」選項中，選擇合適的循環模式。
  • 在這個例子中，選擇「Loop」模式，使動畫無縫循環播放。

8. 設置動畫參數

  • 設置動畫的開始幀(Start Frame)和結束幀(End Frame)，在這個例子中分別設置為0和35。
  • 確保其他參數如速度(Speed)和偏移量(Offset)設置正確，以實現所需的動畫效果。

Step 3.Create a Plane

在Maya中創建一個平面（Plane）。點擊「Create」>「Polygon Primitives」>「Plane」。

Step 4.Adjust Plane Subdivisions

創建MASH網絡並設置分佈類型

1. 選擇MASH選項

  • 在頂部菜單欄中，選擇「FX」>「MASH」>「Create MASH Network」來打開MASH選項對話框。

2. 設置Geometry Type

  • 在MASH Options對話框中，設置「Geometry Type」為 “Mesh”。

3. 設置Distribution Type

  • 在「Distribution Type」選項中，選擇合適的分佈類型。在這個例子中，選擇的是網格分佈。

4. 應用設置

  • 確認所有設置無誤後，點擊「Apply and Close」按鈕應用這些設置並關閉對話框。

Step 5.用MASH工具進行群眾模擬設置

1. 選擇角色組

  • 在Outliner中選擇包含角色的組 (如 “group1” 到 “group5”)。

2. 打開MASH選項

  • 在頂部菜單中選擇「Create」>「MASH」>「Create MASH Network」來打開MASH選項對話框。

3. 設置MASH參數

  • 在MASH選項對話框中，設置「Geometry Type」為 “Instancer”。
  • 選擇「Distribution Type」為網格分佈。

4. 應用設置

  • 點擊「Apply and Close」按鈕以應用設置並關閉對話框。

Step 6.用MASH工具進行群眾模擬設置

1. 選擇MASH網絡

  • 在左側的Outliner面板中，可以看到一個名為 “MASH1_ReproMesh” 和 “MASH2_Instancer” 的MASH網絡。
  • 選中需要編輯的MASH網絡，在這個例子中選擇了 “MASH2_Instancer”。

2. 打開MASH分佈選項

  • 在右側屬性面板中，可以看到MASH分佈節點（MASH_Distribute）的屬性設置。
  • 這裡選擇了「Distribution Type」為 “Mesh”。

3. 設置分佈參數

  • 設置「Number of Points」為 5，表示將有5個實例被分佈。
  • 確認「Distribution Type」設置為 “Mesh”。
  • 確認「Method」設置為 “Scatter”。

4. 連接輸入網格

  • 在「Input Mesh」部分，目前顯示為「Not Connected」。
  • 需要連接一個網格物體作為輸入網格，這將決定實例分佈的位置。

5. 應用設置

  • 完成所有設置後，檢查並應用這些參數。
  • 在右側面板的底部，可以看到「Seed」、「Strength」等額外屬性設置，用於進一步調整分佈效果。

Step 7.使用MASH工具進行群眾模擬設置

1. 選擇MASH網絡

  • 在Outliner面板中，選擇你創建的MASH網絡。在這個例子中，選擇的是 “MASH1”。

2. 打開MASH節點選項

  • 在右側屬性面板中，展開MASH節點選項。你將看到多個可添加的節點類型。

3. 添加Replicator節點

  • 在「Add Node」部分中，選擇並點擊「Replicator」節點。這將幫助你在場景中複製角色，創建群眾效果。

4. 設置節點參數

  • 調整Replicator節點的參數，以控制複製角色的數量和分佈方式。確保角色均勻分佈在指定的區域內。

5. 檢查和調整

  • 在視圖窗口中檢查複製角色的分佈情況，並根據需要進行微調，確保群眾效果符合預期。

Step 8.使用MASH工具進行群眾模擬設置

1. 選擇MASH網絡

  • 在Outliner面板中，選擇你創建的MASH網絡。在這個例子中，選擇的是 “MASH1”。

2. 打開MASH節點選項

  • 在右側屬性面板中，展開MASH節點選項。
  • 點擊「MASH1_Transform」節點，進入變換設置。

3. 設置變換參數

  • 在「Transform」部分，調整位置(Position)、旋轉(Rotation)和縮放(Scale)參數以控制實例的分佈和變換。
  • 點擊「Controller Null」旁邊的「Create」按鈕來創建一個控制器，以便更好地操控變換。

4. 設置強度參數

  • 在「Strength」部分，勾選位置(Position)、旋轉(Rotation)和縮放(Scale)的強度選項。
  • 調整「Strength」、「Random Strength」和「Step Strength」參數，以控制實例變換的隨機性和強度。

5. 應用設置

  • 完成所有設置後，檢查並應用這些參數。
  • 確保「Map Helper」和其他輔助選項設置正確，來進一步調整分佈效果。

Step 9.使用MASH工具進行群眾模擬設置

1. 選擇MASH網絡

  • 在Outliner面板中，選擇你創建的MASH網絡。在這個例子中，選擇的是 “MASH1_ReproMesh”。

2. 打開Deform菜單

  • 在頂部菜單欄中，選擇「Deform」菜單。
  • 展開「Deform」菜單，顯示可用的變形工具選項。

3. 選擇ShrinkWrap變形工具

  • 在「Deform」菜單中，找到並選擇「ShrinkWrap」變形工具。
  • ShrinkWrap工具可以幫助你將實例對象貼合到指定的表面上，以實現更自然的分佈效果。

step 10.使用ShrinkWrap變形工具進行實例貼合

1. 選擇ShrinkWrap變形工具

  • 在右側屬性面板中，選擇你剛才添加的ShrinkWrap節點。在這個例子中，選擇的是 “shrinkWrap1”。

2. 設置投影模式

  • 在「ShrinkWrap Attributes」部分，找到「Projection」選項。
  • 從下拉菜單中選擇「Vertex Normals」作為投影模式。這將根據頂點法線來貼合實例。

3. 調整目標膨脹

  • 調整「Target Inflation」選項，確保實例適當地貼合到目標對象表面。

4. 設置平滑層級

  • 調整「Target Smooth Level」選項，以控制貼合的平滑度。

5. 檢查和調整

  • 在視圖窗口中檢查實例的貼合效果，根據需要進行微調，確保實例正確地貼合到目標對象表面。

step11.使用MASH工具進行ID節點設置

1. 選擇MASH網絡

  • 在Outliner面板中，選擇你創建的MASH網絡。在這個例子中，選擇的是 “MASH2”。

2. 打開Add Node選項

  • 在右側屬性面板中，展開「Add Node」部分。
  • 顯示可用的MASH節點類型。

3. 添加ID節點

  • 在「Add Node」部分，找到並點擊「ID」節點。
  • ID節點可以幫助你為不同的實例分配唯一的標識符，以便進行進一步的控制和變化。
